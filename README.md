# Emotion-detection

The project goal is to detect emotions in user comments. Why do we need a project like this? it is important to the shopping to know if the customer experience was good or not. Besides, it will give them also new ideas to improve their shopping items, experience, solving problems, and etc.

## Project outline
1. importing the library
2. Analizing the data
3. Pre-processing the data
4. Build the model
5. Build a basic website
6. Deploy the model into the website using Flask
